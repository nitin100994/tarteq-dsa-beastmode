## **1. What is an Array? Recap** ğŸ§ 

An **array** is a *collection of items* stored at *contiguous memory locations*.  
It allows us to store multiple values of the **same type** under a **single variable name**.


---

## **2. Key Characteristics** ğŸ”‘

- Arrays are **zero-indexed**: First element is `arr[0]`
- Stored in **contiguous memory**
- **Fixed size** in most languages (not in JS)
- Elements are of the **same data type**

## **3. Common Operations** âš™ï¸

| Operation            | Description                            | Time Complexity |
|----------------------|----------------------------------------|-----------------|
| Access `arr[i]`      | Get element at index `i`               | O(1)            |
| Update `arr[i] = x`  | Update value at index `i`              | O(1)            |
| Insert at end        | Add element at the end                 | O(1)*           |
| Insert at middle     | Insert at position `i`                 | O(n)            |
| Delete at middle     | Remove from position `i`               | O(n)            |
| Search (Linear)      | Look for an element                    | O(n)            |
| Search (Binary)      | Sorted array only                      | O(log n)        |

> *Amortized O(1) for dynamic arrays like JS `push()`*
